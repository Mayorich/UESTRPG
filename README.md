# UESTRPG — Obsidian Vault

Этот репозиторий содержит **vault для Obsidian** с переводом и структурированными заметками по системе **UESRPG**  
(адаптация правил в формате, удобном для навигации, ссылок и дальнейшего расширения).

Проект ориентирован на использование **в Obsidian** как:
- справочника правил,
- базы знаний для ведущего,
- основы для собственной кампании или мира.

---

## Как пользоваться

1. Склонируйте репозиторий:
   bash
   git clone https://github.com/Mayorich/UESTRPG.git
   
2. Откройте папку как vault в Obsidian

3. Включи Community Plugins, список плагинов хранится в .obsidian/plugins/.
